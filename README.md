# Axis
minimalist mission control for daily stats. track what you eat, how you move, habits, and long-term data in one place with zero friction.

### tools
- **habits**: grid tracking with shift-drag batch checks and streaks.
- **food**: calorie and protein logging with historical archive.
- **move**: categorized workout folders with persistent daily notes.
- **vault**: persistent media gallery with favoriting and metadata.
- **tasks**: high-speed daily checklists and goals.

### data
- **charts**: longitudinal trends for nutrition and consistency.
- **correlation**: protein vs calorie scatterplots for energy density.
- **heatmap**: 12-week workout frequency grid with log drill-down.
- **flow**: moon-arc time visualization and midnight countdown.

dark mode by default. fast. no fluff.
